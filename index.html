Temperature Conversion

This Ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

      puts "Please enter degrees in Celcius:"
        
      c = gets.chomp.to_f

      def c_to_f(x)
        return x * 1.8 + 32.0
      end

      f = c_to_f(c)

      puts "The temperature is #{f} degrees Fahrenheit."

      puts "Please enter a pleasant number, one that sounds
      nice as it rolls off the tongue."

      number = gets.chomp.to_i

      def suffix(number)
        modulo = number % 10

        if number == 11 || number == 12 || number == 13
          return "th"
        elsif modulo == 1
          return "st"
        elsif modulo == 2
          return "nd"
        elsif modulo == 3
          return "rd"
        else
          return "th"
        end
      end

      puts "You know, what a funny coincidence --
      You're the #{number}#{suffix(number)} person to
      give me that answer!"

Foobar

This Ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

      puts "Let's play the FooBar game! How many numbers should I do?"

      answer = gets.chomp.to_i

      puts "Okay! Here goes:"

      answer.times do |n|
        iteration = n + 1

        if iteration % 3 == 0 && iteration % 5 == 0
          puts "Foobar"
        elsif iteration % 3 == 0
          puts "Foo"
        elsif iteration % 5 == 0
          puts "Bar"
        else
          puts iteration
        end
      end

      puts "Yay! That was fun!"
